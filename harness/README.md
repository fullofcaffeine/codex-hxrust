# Harness

Fixture harness work starts here.

Initial fixture-run schema:

```json
{
  "schema": "codex-hxrust.fixture-run.v1",
  "id": "upstream.basic-one-turn",
  "source": "upstream-codex",
  "kind": "sse-sequence",
  "input": {
    "events": []
  },
  "expect": {
    "notifications": [],
    "state": {}
  }
}
```

Keep harness inputs credential-free.
